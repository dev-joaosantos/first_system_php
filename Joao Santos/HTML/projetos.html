Projetos Acadêmicos com HTML & CSS

Projeto Pandora

O Projeto Pandora é um design web desenvolvido com o objetivo de ilustrar um e-commerce voltado para tutores de porquinhos da índia. Este projeto foi criado em dupla como parte do meu portfólio para demonstrar minhas habilidades em design e desenvolvimento web. A ideia é proporcionar uma experiência de compra agradável e intuitiva para os usuários, com um layout visualmente atraente para os usuários que visualizaram o Projeto Pandora.

Projeto Lajoá

O Projeto Lajoá é um design web desenvolvido com o objetivo de ilustrar um e-commerce voltado para amantes de açaí. Este projeto foi criado em dupla como parte do meu portfólio para demonstrar minhas habilidades em design e desenvolvimento web. A ideia é proporcionar uma experiência agradável ao usuário e instigar a vontade de consumir os produtos oferecidos. O layout é responsivo e visualmente atraente, visando atrair usuários que visualizaram.
